#MlLex and MlYacc
## Dependencies
* mlton compiler

## Build executable
```bash
make
```
This builds the executable ./a2

## Run
```bash
./a2 [arg]
```
Here, [arg] is the name of the input file.

## Usage
* Build the executable
* Run the executable as
```bash
./a2 input.txt
```
where input.txt is a sample input file.
